Integration Setup

  1. Clone: git clone https://github.com/kdairatchi/claude-bug-bounty ~/claude-bug-bounty
  2. Copy template: cp templates/hunter.yaml.example ~/claude-bug-bounty/config/hunter.yaml
  3. Add env vars to ~/.zshrc using the Shell Template above
  4. Run: bash ~/claude-bug-bounty/install.sh
  5. Verify: python3 ~/claude-bug-bounty/tools/doctor.py --section env
  1. Install: go install -v github.com/projectdiscovery/subfinder/v2/cmd/subfinder@latest
  2. Create config dir: mkdir -p ~/.config/subfinder
  3. Copy the subfinder block above to: ~/.config/subfinder/provider-config.yaml
  4. Fill in your actual keys (remove angle brackets)
  5. Test: subfinder -d example.com -all -v 2>&1 | grep "source" — verify sources appear
  1. Install: pip3 install waymore
  2. Config file: ~/.config/waymore/config.yml
  3. Add keys:
    URLSCAN_API_KEY: <YOUR_URLSCAN_KEY>
VIRUSTOTAL_API_KEY: <YOUR_VT_KEY>
  4. Test: waymore -i example.com -mode U
  1. Install: go install -v github.com/projectdiscovery/nuclei/v3/cmd/nuclei@latest
  2. No API key needed for core scanning
  3. Update templates: nuclei -update-templates
  4. PDCP integration (optional): set CHAOS_KEY and use nuclei -cloud-list
  5. Test: echo "https://example.com" | nuclei -t http/technologies/ -silent
